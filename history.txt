  Rule established: only categories with real content get added.

    macOS     → Apple logo, gold (#fbbf24)

- library.html + os-fundamentals.html: Changed sub-description text from
  inline bullet format ("Linux · Windows · macOS · Android") to stacked
  vertical list (each item on its own line). Cards now grow in height
  naturally with more items.

- library.html + os-fundamentals.html: Removed all Active/Soon badge
  elements and their CSS entirely.

- index.html: Removed fake search bar (HTML, CSS, and JS all stripped).
  A real search bar will be added later when ready.

- index.html — Cursor animation: Fixed lag by switching from left/top
  property animation to transform: translate() (GPU compositor, no layout
  recalc). Added will-change: transform. Dot now snaps exact, ring trails.

- index.html — Title hover animation: Replaced laggy titleGlitch (was
  animating text-shadow + letter-spacing = forced paint/layout) with
  titleFlare (transform: scale + filter: brightness/drop-shadow = smooth,
  compositor-only glow+grow effect).
